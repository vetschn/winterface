Detailed Description

template<class MT, class AT>
class ll__::mat_b< MT, AT >

Wrapper for a matrix.

The api exposing const access is public. The api exposing all other access is public. This class is meant to be inherited from for classes who are essentially wrappers of a matrix. Some of the api is directly exposed such that we can write A.ccBegin() instead of A.mat_.ccBegin() without having to write a pass- through manually.
