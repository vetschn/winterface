Detailed Description

template<class MT, class AT, class CT, class CB>
class ll__::mat_vec_cb< MT, AT, CT, CB >

Wrapper for a class holding a matrix and vector of the same number of columns/elements.

The api exposing const access is public. The api exposing all other access is protected. Some of the api is directly exposed such that we can write A.cbegin() instead of A.vec_.cbegin() without having to write a passthrough manually. Some of the functions are such that the operation is done on both containers simultaneously. Both containers must be of the same size at all times.
