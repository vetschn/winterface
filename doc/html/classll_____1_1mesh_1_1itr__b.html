Detailed Description

template<class IT, class RT, class RT_>
class ll__::mesh::itr_b< IT, RT, RT_ >

mesh iterator base class.

A mesh iterator traverses a mesh along a 'line' in the mesh, which is implemented by a matrix underneath. This matrix can be traversed column-wise in a manner similar to along columns or rows with respect to the underlyng linear data. Since a matrix is essentially a 2D grid, it can be traversed along rows or columns (or 'lines') by an iterator with a custom step. A mesh iterator works analogous to lm_tItr with a custom step hopping from column to column along 'lines' in the mesh.
Template arguments are:

  • IT: column iterator
  • RT: return type
  • RT_: other return type
