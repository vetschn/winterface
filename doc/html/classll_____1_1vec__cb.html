Detailed Description

template<class CT, class CB>
class ll__::vec_cb< CT, CB >

Wrapper for a const vector (std::vector or otherwise).

The api exposing const access is public. The api exposing all other access is protected. This class is meant to be inherited from for classes who are essentially wrappers of a vector. Some of the api is directly exposed such that we can write A.cbegin() instead of A.vec_.cbegin() without having to write a pass- through manually.
