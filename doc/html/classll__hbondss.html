Detailed Description

multiple ll_hbonds.

This class is designed with a broadly equivalent interface as 'll_hbonds' and as such either can be used in many contexts. The generalized bond indices used here hold the wbh index in the first byte and the regular index in the bytes above. The naked numbers therefore appear to not make much sense. In any case, this class can be used without adding significant overhead compared to ll_hbonds or even as a complete replacement for the case of just one wbh. The approach here is required when generating Hamiltonian matrices from multiple Wannierizations.
