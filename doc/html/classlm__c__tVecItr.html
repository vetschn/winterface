Detailed Description

template<class MT, class VT>
class lm_c_tVecItr< MT, VT >

Vector const_iterator class.

Can be constructed from other vector iterators or const_iterators. Const Correctness is ensured by returning const rows or columns when dereferencing thereby making changes in the 'host' matrix impossible.
