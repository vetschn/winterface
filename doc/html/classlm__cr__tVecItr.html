Detailed Description

template<class MT, class VT>
class lm_cr_tVecItr< MT, VT >

Reverse vector const_iterator class.

Can be constructed from other (reverse) vector iterators or const_iterators. Const Correctness is ensured by returning const rows or columns when dereferencing thereby making changes in the 'host' matrix impossible.
