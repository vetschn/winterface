Detailed Description

template<class TT, class FT, class CT>
class lm_tArray< TT, FT, CT >

The parent class for matrices, rows and columns.

Implements all features common to matrices, rows and columns, thus enabling polymorphism between these types. Whenever treatment as an array is sufficient, this class should be used over its children. However it is important to realize that the main class is the matrix, since rows and columns cannot exist independently and are hosted by a 'host' matrix where they represent rows or columns of this matrix. However standalone rows or columns are still possible in the form of an appropriately shaped matrix. The number of rows is referred to as M, the number of columns as N, the linear size, i.e. the total number of elements, as L. Since the underlying data is arranged COLUMN-MAJOR (due to calls to FORTRAN routines), working over columns should be preferred to over rows wherever possible. Template arguments are:

  • TT: the underlying type of this, either real (if this is an fArray) or complex (if this is a cArray)
  • FT: the real type
  • CT: the complex type
