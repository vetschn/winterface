void ll__::balance ( wi &  I,
const ll_cell &  cell,
const fMat &  Wp,
const fMat &  NN 
)
noexcept

function to 'balance' out a Wannier matching.

This can be useful for suboptimal Wannierizations where some Wannier centers can be a bit further away than usual. They may then get matched to the 'wrong' atom. This function attempts to balance things out by shifting the least obvious matches such that the same atomic types all have the same number of Wannier centers matched to them despite the spacial warping of the initial distribution.

cellunit cell
