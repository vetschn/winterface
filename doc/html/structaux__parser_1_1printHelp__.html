Detailed Description

Print help struct.

This struct defines how help messages for each key should be printed. Each child of aux_parser must provide their own version of this struct which must inherit from this class. The printing process is defined in the constructor, who will then call all the other constructors in the inheritance hierarchy from top to bottom, thus printing all the messages. Additionally, the current contents of a child of aux_parser will be printed. Thus when not initialized, the default values will be printed. The recognized types can be extended by writing additional print_ functions.
